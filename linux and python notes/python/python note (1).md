# 2. python \_\_name__属性

当一个 Python 脚本作为主文件或者主模块被直接运行时，"\_\_name__" 的值就是 "\_\_main__"。

而当该脚本被其他模块导入时，即"\_\_name__"位于次文件（被导入的文件）中，则执行"\_\_name__"时返回的值就是导入的文件名。
